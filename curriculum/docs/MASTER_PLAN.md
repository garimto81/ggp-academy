   **※ IA (Immediate Analysis)란?**

   - B급: 의미있는 액션 (큰 팟, 중요한 폴드) → 대기열
   - C급: 일반 핸드 (빠른 폴드, 작은 팟) → Skip (아카이브만)

   2. 핸드 분석 (5분 이내)

   4. CUE SHEET 업데이트
   5. A급 핸드를 Editor 팀에 즉시 전달

   **효과:**
   - 편집 대상을 전체의 10-20%로 축소
   - 1~2시간 내 송출 가능한 워크플로우 구현
   - 고품질 콘텐츠만 선별 송출

      - 핵심 액션만 타이트하게 편집

   b) 샷 순서 조정 (10분/핸드)

   [3단계: 2D - 자막 및 그래픽 작업 (촬영 후 60~70분)]
