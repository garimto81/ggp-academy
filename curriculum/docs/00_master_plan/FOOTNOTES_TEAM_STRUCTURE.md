### 📡 RFID Operator

**소속**: 외주 스탭 (이벤트별 계약)

**인원**: 피처 테이블당 1명

#### 주요 역할
1. **카드/칩 리더 입력**
   - RFID 카드 리더 운영
   - 칩 카운트 수동 입력 (RFID 미지원 시)
   - 베팅 액션 실시간 입력

2. **데이터 정합성 검증**
   - 입력 데이터 정확성 체크
   - 오류 발생 시 즉시 수정
   - 카메라 팀에 데이터 전달

3. **장비 관리**
   - RFID 리더기 세팅
   - 배터리 및 연결 상태 모니터링
   - 기술 파트와 장비 이슈 협의

#### 🔗 데이터 흐름
```
RFID Operator 입력 → POKER GFX 시스템 → 실시간 그래픽 출력 → IA 팀 전달
```
